# kali 窗口不能移动或没有了右上角的控制按钮拯救计划

今天晚上，月色明媚，我在电脑前一边吃火锅、唱歌，一边编写Wave的WP。写到一半时，我发现虚拟机有些异常。我就注销了系统。然而，当我重新登录后，发现终端窗口无法移动，右上角的控制按钮不见了，任务栏图标也消失了——显然，事情并不如预期。

## 临时解决方法

幸运的是，我找到了以下临时解决办法：

```bash
xfwm4 --replace &
```
